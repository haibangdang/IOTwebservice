Database:
- Cài Postgresql
- Tạo db nhu file dbconfig.js (Các table trong DB sẽ được gen khi create workspace)
- Create schema cần thiết (dùng API create /api/workspace/create)
Môi trường:
- Install nodejs
- Install npm
- Install express: npm install express -save
- Install các thư viện còn lại khi chạy k được: npm install <tên thư viện> -save
Start server
- node server.js
